A quick configurable Instagram image feeds shortcode. 

== Description ==
This plugin is developer and distributed as free. If you have any issue or query please feel free to contact me.

use this shortcode [cp-instagram view="json"] for JSON output or simply use  [cp-instagram] for html output.

== Installation ==

1. Upload the plugin and activate.
2. You will get a settings page in the admin section.
3. Add instagram key and secret and copy the URL from the address bar and paste in URL field.
4. Click the Authenticate link to authenticate your app.
5. Checked the check boxes as per your requirements.
